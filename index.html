Hey, there! Why are you stalking me? I'm a brazilian systems analist, translator and, sometimes, a programming languages designer. Let me introduce myself to you!

Keep Code Quality

  • Static Typing

    Have control and type security with a Haskell-like type system and types as objects.

  • Functional Programming

    Function-composition, pattern-matching, monads abd algebraic data-types.

  • Pure Object Orientation

    A more pure and Smalltalk-like object orientaton, where everything, including types, is an object.

  • Computations May Fail

    Treat errors as first-class, and abstract over failure flows with the Maybe(a) and Either(a, b) monads.

  • Prototype Based

    Supports a Self/Javascript based prototype programming, in a much more expressive way than class-based-programming does.

  • First Class Data Types

    Types are all first-class citizens and can be derived or reimplemented in your proper manner. Simple, but powerful typing by wrapping.
